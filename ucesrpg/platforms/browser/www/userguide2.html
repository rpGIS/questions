Quiz Input Form User Guide

Here is where you enter the questions for the quiz

Start by selecting a location for the question by dragging the marker and dropping it at your location of choice

Fill out the question box with your question and then select four potential answers

Select which answer is correct using the drop down menu

Note: You must fill in all boxes before submitting, your question must have four answer choices!

Once you have filled in all the information, press the Upload Question button to upload to the server, the page will be refreshed at this point and the contents of the form will be uploaded to the database

You can click the button to go to my github repository page, where all the code used to make this webapp, and the corresponding phone app, can be found
